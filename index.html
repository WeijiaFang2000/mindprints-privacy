MindPrints 隐私政策

最后更新：2026 年 3 月

MindPrints 由个人开发者开发。本政策说明本应用如何获取和处理您的数据。我们不运营任何服务器，也不收集您的任何数据。所有信息均存储在您的设备本地。

1. 相册访问

本应用会请求读取您的相册，仅用于导入您主动选择的截图。我们不会读取、上传或访问您未选择的任何照片。

导入完成后，您可以选择删除相册中的原始截图。删除操作仅针对您在导入流程中明确选中的图片，由系统 Photos API 执行。

2. 麦克风与语音识别

Archive 搜索页提供语音输入功能。使用时，本应用会请求麦克风权限，并使用 Apple 系统的语音识别框架（SFSpeechRecognizer）将语音转为文字。语音数据由 Apple 处理，适用 Apple 隐私政策。我们不保留任何录音。

3. 本地通知

本应用发送以下类型的本地通知：

  • 「可以试试」提醒：您离开应用约 90 分钟后，提示回顾某条待跟进的屏踪
  • 每日定时提醒：每天 1–2 次，提示关注某个主题
  • 周洞察生成完成通知

所有通知完全在设备本地调度，不经过任何服务器。您可随时在系统设置中关闭通知。

4. 本地 OCR 识别

导入截图时，本应用使用 Apple 的 Vision 框架在设备本地对图片进行文字识别（OCR），用于过滤验证码、取件码、密码等敏感临时内容，并生成可搜索的文字索引。OCR 全程在本地运行，识别内容不会上传至任何外部服务。

5. AI 功能与 OpenAI

本应用的 AI 搜索、摘要生成、洞察报告等功能需要您自行提供 OpenAI API Key。配置后，相关截图的图像和文字内容将通过您的 API Key 发送至 OpenAI 处理。

我们对通过您的 API Key 发送至 OpenAI 的数据不做任何收集或转存。OpenAI 的隐私政策适用于该过程：openai.com/privacy

未配置 API Key 时，所有功能均在本地运行，不涉及任何外部网络请求。

6. 数据存储

本应用将以下数据存储在您的设备本地：

  • 导入的截图图片文件（存储于应用沙盒 Documents 目录）
  • 备注、标签、AI 生成的摘要与洞察（JSON 格式，存储于本地）
  • 应用设置与偏好
  • Widget 所需的共享数据（通过 App Group 存储于设备本地）
  • OpenAI API Key（使用系统 Keychain 加密存储）

以上所有数据仅存在于您的设备，我们无法访问。

7. 数据删除

在应用内删除任意屏踪，数据将立即从设备移除。卸载本应用将删除所有本地数据（系统 Keychain 条目除外，可在设置中手动清除）。

8. 儿童隐私

本应用不面向 13 岁以下儿童，也不会有意收集儿童个人信息。

9. 政策变更

如本政策发生重大变更，我们将通过应用更新日志告知用户，并更新本页面顶部的日期。

10. 联系方式

如有疑问，请通过 App Store 页面的开发者联系方式与我们联系。

MindPrints Privacy Policy

Last updated: March 2026

MindPrints is developed by an independent developer. This policy explains how the app accesses and handles your data. We do not operate any servers and do not collect any of your data. All information is stored locally on your device.

1. Photo Library Access

The app requests access to your photo library solely to import screenshots you explicitly select. We do not read, upload, or access any photos you have not chosen.

After import, you may optionally delete the original screenshots from your photo library. Deletion applies only to images you explicitly selected during the import flow, executed via the system Photos API.

2. Microphone & Speech Recognition

The Archive screen offers a voice input feature for search. When used, the app requests microphone access and uses Apple's speech recognition framework (SFSpeechRecognizer) to transcribe speech to text. Audio is processed by Apple and subject to Apple's privacy policy. We do not retain any audio recordings.

3. Local Notifications

The app may send the following types of local notifications:

  • Resurface nudges: sent ~90 minutes after you leave the app, prompting you to revisit a pending item
  • Daily reminders: 1–2 per day, tied to specific time windows
  • Weekly insight ready: notifies when your weekly reflection has been generated

All notifications are scheduled entirely on-device and do not pass through any server. You can disable them at any time in iOS Settings.

4. On-Device OCR

When importing screenshots, the app uses Apple's Vision framework to perform optical character recognition (OCR) locally on your device. This is used to filter out sensitive temporary content (e.g. verification codes, passwords) and to build a searchable text index. OCR runs entirely on-device; recognized text is never uploaded to any external service.

5. AI Features & OpenAI

AI-powered search, summarization, and weekly insights require you to provide your own OpenAI API Key. Once configured, relevant screenshot content (images and extracted text) is sent to OpenAI via your API Key for processing.

We do not collect, store, or relay any data sent to OpenAI through your API Key. OpenAI's privacy policy governs that processing: openai.com/privacy

If no API Key is configured, all features run locally with no external network requests.

6. Data Storage

The app stores the following data locally on your device:

  • Imported screenshot image files (in the app's sandboxed Documents directory)
  • Notes, tags, and AI-generated summaries and insights (stored as local JSON)
  • App settings and preferences
  • Widget data shared via App Group (stored locally on-device)
  • Your OpenAI API Key (encrypted in the system Keychain)

All of the above exists only on your device. We have no ability to access it.

7. Data Deletion

Deleting any item within the app removes it from your device immediately. Uninstalling the app removes all local data (Keychain entries may persist and can be cleared manually via Settings).

8. Children's Privacy

MindPrints is not directed at children under 13 and does not knowingly collect personal information from children.

9. Changes to This Policy

If we make material changes to this policy, we will notify users through app release notes and update the date at the top of this page.

10. Contact

For questions or concerns, please reach out via the developer contact on the App Store listing.
